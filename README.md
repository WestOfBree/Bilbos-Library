## Bilbo's Library
A React book store project built for class.

This website was a class project showcasing a simple ecommerce site. The prompt we were given was to build a book store app. Assets were provided (logo + illustrations), along with book data to display across the site. I however went a step further from the original class assets by curating my own in-app book data to match the theme and style of Bilbo's Library.

## Tech Used

- React 19
- React Router DOM 6
- JavaScript (ES6+)
- HTML5 + CSS3
- Font Awesome React
- Create React App tooling (`react-scripts`)

## Skills Demonstrated

- Building a multi-page SPA with client-side routing
- Component-based architecture and reusable UI patterns
- State management with React hooks (`useState`, `useEffect`)
- Cart functionality (add, remove, quantity updates, totals/tax calculations)
- Data rendering and filtering/sorting logic (price low-high, high-low, rating)
- Conditional UI rendering (empty cart state, in-cart state, fallbacks)
- Clean layout structure and responsive front-end styling

## Core Features

- Home page with hero/highlights/featured sections
- Books page with sorting controls
- Single book details page with recommendations
- Shopping cart page with dynamic totals
- Simple checkout placeholder flow


## Project Context

This project was focused on applying front-end fundamentals in a practical ecommerce build: routing, reusable components, and interactive state-driven UI.
